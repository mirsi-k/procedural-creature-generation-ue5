# procedural-creature-generation-ue5
Unreal Engine 5 Blueprint prototype for procedural creature generation.

This repository contains the Unreal Engine 5 prototype developed for the thesis:

**Σύστημα Διαδικαστικής Δημιουργίας Πλασμάτων για Δυναμικά και Ποικιλόμορφα Οικοσυστήματα Παιχνιδιών**

The prototype is implemented mainly through Unreal Engine 5 Blueprint visual scripting. It uses predefined modular creature assets, such as bodies, heads, legs, tails, and horns, and combines them through controlled random selection and basic assembly rules.

## Notes

This prototype does not include full rigging, advanced animation, artificial intelligence, or procedural generation of new geometry. It focuses on modular assembly and controlled variation of predefined creature parts.
